Description
===========

Quad SPI flash library.

Key Features
============

 * Supports via API
     * read,
     * write,
     * erase,
     * access to status registers.
 * Runs as a library call, requiring no extra cores or timers.
 * Up to 23.8 MB/s read speed.
 * Includes application demonstrations
