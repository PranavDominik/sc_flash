Quad SPI Flash Demo
===================

:scope: General Use
:description: Quad SPI flash
:keywords: quad,spi,flash 
:boards: XA-SK-FLASH

Features
--------

 The quad SPI flash component has the following features:
  * Supports via API
     * read,
     * write,
     * erase,
     * access to status registers.
  * Runs as a library call, requiring no extra cores or timers.
